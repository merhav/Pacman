Welcome to our about page!

In case you haven't noticed yet, for the third time, the website is developed by Ido Marko(209510890) & Itay Merhav(205503352).

The JQuery we used was from a Google API.
The link we used was the following: https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js
We used jQuery in our websites to make selection of diffrent objects, trigger events and valdidate the different form.

There were a few difficulties while programming the website, But the one that felt the most was making the ghosts appear on the canvas and programming them in a way that they won't eat the food.
We had a few ideas on solving this issue but we had 2 main one. The first was to put a canvas on a canvas, seprating the ghosts from the food but that made a real mess with the footer for some reason. The second idea was to give different code names to places where ghosts were on food. So for example if ghost 1 was on food 2, The code of this square would be "12".
